# -Kelly-Roland-Coffee-Shops-2023-Operational-Report
Project: Kelly Roland Coffee Shops - 2023 Operational Report Objective: To analyze the 2023 operational performance of Kelly Roland Coffee Shops by identifying key sales trends, peak performance periods, and product profitability to provide actionable insights for strategic decision-making.
Data & Tools: Utilized a transactional sales dataset sourced from Kaggle. The entire analysis, including data cleaning, transformation, and visualization, was performed using Microsoft Excel. Key features included PivotTables, advanced formulas (XLOOKUP, SUMIFS), and interactive dashboards.
